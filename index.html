⚙️ Instrucciones de Uso

  1. Descarga el archivo .ipynb haciendo clic en el botón
  2. Abre el notebook en Google Colab o Jupyter
  3. Instala las dependencias: pip install streamlit tensorflow scikit-learn plotly
  4. Ejecuta las celdas en orden
  5. Para Streamlit: streamlit run app.py
